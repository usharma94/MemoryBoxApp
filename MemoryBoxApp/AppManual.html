MemoryBox

Welcome to the MemoryBoxApp. A simple solution for creating and sharing memories.

On the home page, select new memory. You will be directed to the add new memory form. Here you can add the name, description, time, location, and photo of your memory.

Once you have saved a memory you can view it in the memory list. All memories can be viewed in a list. Access this list from the home page. You have the ability to futher view memory details by selecting the memory.

From home page select view memories. Click on the memory you would like to share. Click share memory. You will see a list of users. Select user you would like to share memory with.

Under home page select Update Account. You will be naviaged to a form in which you can update user information. Once done, click update button.
